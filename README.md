### Project setup
```

- Install dependencies
```
npm install
```

- Compiles and hot-reloads for development
```
npm run serve
```

- 线上打包
```
npm run build
```

- 开发环境打包
```
yarn run dev
```

- 预发环境打包
```
yarn run staging
```
